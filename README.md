# TreeCalc-99

This project uses a binary expression tree to store mathematical data for conversion and evaluation between prefix, infix, and postfix notations using postorder DFS to calculate height and node count.

How to Use:
1. Compile using GCC: gcc main.c functions.c -o calculator
2. Run the executable: ./calculator (or calculator.exe on Windows).
3. Select Option 1 from Menu.
4. Enter a Postfix Expression.
5. Use Option 2, 3, 4 for Prefix, Infix, or Postfix.
6. Select Option 5 to evaluate using DFS.
7. Selection Option 6 to display results.

Quit anytime by selecint Option 8.
